# 策略生成 Prompt 说明

这个文件不是直接执行的代码，而是保险顾问 Skill 的策略生成 Prompt 说明。

## 输入

- 压缩后的客户上下文；
- 从业者画像；
- Sales Intelligence digest；
- 可选新闻摘要；
- 合规规则。

## 输出

- 当前客户推进判断；
- 低压沟通策略；
- 可直接使用的话术；
- 客户不同反应的接法；
- 下一步动作；
- 最多两个仍待确认的问题。

## 硬规则

- 不承诺收益；
- 不承诺避税避债；
- 不制造恐惧；
- 不编造案例；
- 不贬低其他金融产品；
- 涉及产品时必须中性、克制、合同条款优先。
